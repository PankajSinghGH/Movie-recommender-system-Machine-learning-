# Movie-recommender-system-Machine-learning-

## Overview
This project is a content-based movie recommendation system built using Machine Learning techniques. It suggests movies to users based on similarity in content such as genre, keywords, and other metadata.

The system uses Natural Language Processing (NLP) techniques to transform text data into numerical form and compute similarity between movies.

---

## Features
- Recommend similar movies based on user input
- Uses TF-IDF vectorization for text processing
- Computes similarity using cosine similarity
- Simple and interactive interface using Flask
- Fast and efficient recommendation system

---

## Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy
- Flask (for web interface)

---

## How It Works
1. Movie dataset is preprocessed and cleaned
2. Important features like genre, overview, and keywords are combined
3. TF-IDF Vectorizer converts text into numerical vectors
4. Cosine similarity is used to find similarity between movies
5. Top similar movies are recommended to the user
